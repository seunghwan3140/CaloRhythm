# ‚ÑπÔ∏è About CaloRhythm

**CaloRhythm** is an intelligent, open-source nutrition management system  
designed specifically for the **Korean dietary environment**.  
It combines official government nutrition data with mathematical optimization  
to help users build a healthy and sustainable dietary rhythm.

---

## üéØ Purpose

Modern lifestyles offer convenience but often lead to nutritional imbalance.  
Many people lack tools that provide *accurate, Korea-specific* nutritional insight.

CaloRhythm aims to solve this by:

- Using the **National Standard Food Composition Database (v10.3)**  
- Evaluating meals against **KFDA Adult Daily Nutrient Standards**  
- Providing algorithm-driven nutrient analysis and optimization  
- Helping users discover healthier alternatives through nutrient-density ranking

---

## üõ† Technology Stack

CaloRhythm is built as a streamlined, single-page application (SPA) using:

- **Python 3.9+**  
- **Streamlit** (UI Framework)  
- **Pandas / NumPy** (Data processing and computation)  
- **SciPy SLSQP** (Optimization engine)  
- **Altair** (Statistical visualization)  
- **OpenPyXL** (Excel-based data ingestion)  
- **Sphinx + ReadTheDocs** (Documentation)

---

## üìÇ Data Source

The system relies on the **Korean National Standard Food Composition Database (Version 10.3)**,  
provided by the Rural Development Administration (RDA).  
All nutrient calculations are based on standardized 100g servings from the official dataset.
