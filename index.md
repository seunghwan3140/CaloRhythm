An intelligent, open-source nutrition management system optimized for the **Korean dietary environment**.

CaloRhythm uses the **National Standard Food Composition Database (v10.3)** and  
mathematical optimization algorithms (SciPy SLSQP) to deliver highly accurate nutrient analysis,  
portion optimization, and nutrient-density-based food recommendations.

## ðŸ”§ Core Features Overview
- **Standard Nutrition Calculator**  
  Calculates total nutrient intake and compares it against **KFDA Daily Standards**  
  using official Korean nutritional data (RDA v10.3).

- **AI Diet Optimizer (SLSQP)**  
  Computes mathematically optimal ingredient portions based on user limits,  
  priority mode, and minimum intake constraints.

- **Food Discovery (Nutrient Density Ranking)**  
  Identifies the most nutrient-dense or nutrient-light foods using statistical sorting  
  (Top N / Bottom N by selected nutrient).
